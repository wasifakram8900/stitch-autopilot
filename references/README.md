# references/

Drop the 100 "greatest website" `.md` files here (one per file). The **Reference Scout**
(next build) auto-tags each by niche + style on load, then picks the best matches per business
to steer the Design Director.

No strict format required — plain Markdown describing a site is fine. If you want to help the
Scout, you can start a file with a tag line, e.g.:

    <!-- niche: dental, gym; style: bold, dark, kinetic -->

Otherwise the Scout infers tags from the content.
