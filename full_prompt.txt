Build a complete, premium single-page local business website WITH an integrated booking system as ONE single index.html file. One file. Everything inside it. No separate pages. No external JS libraries except Tailwind CDN and Lucide Icons CDN.

The website has two views toggled by JavaScript:
  - Home view  -> id="page-home"
  - Booking view -> id="page-book"

Switching is done with this function (implement it exactly):
  function showPage(id) {
    document.querySelectorAll('.page-view').forEach(p => p.style.display = 'none');
    document.getElementById('page-' + id).style.display = 'block';
    window.scrollTo({ top: 0, behavior: 'smooth' });
  }
"Book Now"/"Book Appointment"/"Book This" -> all call showPage('book'). "Back to Home" -> showPage('home'). No href="#". No alert(). No page reload. No separate HTML files.
